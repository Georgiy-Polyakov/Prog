## Циклы

#### Оператор <code>while</code>

#### Цикл с предусловием
```c
while (условие_выполнения_цикла)
{
    операторы...
    [break;]
    [continue;]
}
```

#### Цикл с постусловием
```c
do
{
    операторы...
    [break;]
    [continue;]
}
while (условие_выполнения_цикла);
```

> В цикле <code>while</code> важно, чтобы хотя бы одна переменная условия цикла менялась в теле цикла, иначе цикл может не закончиться.
