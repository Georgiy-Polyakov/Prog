## Циклы

#### Оператор <code>for</code>

```c
for (раздел_объявлений; раздел_условия; раздей_действий)
{
    операторы...
    [break;]
    [continue;]
}

раздел_объявлений - содержит создание и инициализацию переменных цикла  
раздел_условия - содержит условие выполнения цикла (Когда оно нарушается, цикл заканчивается).  
раздел_действий - содержит действия над переменными цикла  

break - прекращает выполнение цикла  
continue - прерывает выполнение цикла и переходит к следующей итерации (итерация - один проход цикла)  
```
